Common Questions About Bulk Delete

How to delete all Gemini chats at once?

Install the Gemini Bulk Delete extension, toggle on bulk delete mode, click "Select All" to choose all chats, then click "Delete Selected". All your chats will be removed in seconds.

Can I select which chats to keep?

Yes! You can manually check/uncheck specific chats. Use "Select All" then uncheck the ones you want to keep, or individually select only the chats you want to delete.

Is there an undo option?

No, deletions are permanent. The extension shows a confirmation dialog before deleting to prevent accidents. Make sure to export important chats as PDFs before bulk deleting.

How many chats can I delete at once?

There's no limit! Users have successfully deleted 100+ chats in a single operation. The extension handles any number of chats efficiently.

Does it work with Google AI Studio?

Currently, the bulk delete feature works only with gemini.google.com. AI Studio support may be added in future updates.
